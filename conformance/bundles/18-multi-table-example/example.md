# A basket imports users and assets

Given the following users have been imported:

| email | name |
| ----- | ---- |
| a@b.c | Ada  |

And the following assets have been imported:

| name  |
| ----- |
| Moose |

Then the basket contains 1 user and 1 asset.

A basket starts empty, and this prose sentence separates the two examples.

Then the basket contains 0 users and 0 assets.
