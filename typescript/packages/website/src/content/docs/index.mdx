Entropy is everywhere. Code and tests tend to drift apart from documentation and specifications in subtle ways.
Vár closes that gap by making your documentation *executable*.
When a programmer or coding agent breaks the oath, Vár catches it. Every time.

## Examples

The examples below give a taste of how Vár works. If the markdown disagrees with the code, your tests will fail.

## Who is Vár?

In Norse mythology, Vár is a goddess who listens to people's oaths and private agreements. 
She punishes those who break them.[^1]
