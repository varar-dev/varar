---
title: Oaths
description: A markdown document with checkable claims.
---

An *oath* is a Markdown document with one or more tests.

A Markdown document is an oath if it's linked to your code through [sensors](/reference/sensors) and [stimuli](/reference/stimuli).

It's tedious to talk about "executable markdown", "executable specifications", "executable documentation",
"markdown tests" etc, so we've chosen this simple term to distinguish from regular Markdown documents *without* tests.
